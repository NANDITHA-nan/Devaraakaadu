# 🌿 DEVARA-KAADU
### Guardians of the Green Temple

DEVARA-KAADU is an Android application designed to promote awareness, conservation, and digital interaction with India's sacred groves. The app combines ecological education, mythology, biodiversity scanning, incident reporting, and grove exploration into a modern mobile experience.


## 📱 Features

### 🌳 Grove Directory
Explore sacred groves across India with detailed information including:
- Grove names
- Ecological significance
- Traditional importance
- Sacred classifications


### 🗺 Interactive Map
Locate sacred groves geographically using map integration.

Features:
- Grove location visualization
- Geographic exploration
- Sacred grove discovery



### 📖 Myths & Legends
Discover cultural stories and scientific interpretations of sacred forests.

Included groves:
- Iruppu Sacred Grove
- Tamhini Ghat Devrai
- Khajurli Oran
- Hulikal Bana
- Sarna Sacred Grove
- Nagarahole Devara Kaadu

Features:
- Sacred myth descriptions
- Ecological reality explanations
- Traditional observances
- Scientific observations
- Scrollable lore exploration



### 📷 Species Scan
AI-inspired biodiversity scanning module.

Users can:
- Open live camera
- Capture flora/fauna images
- Upload gallery images
- Simulate species recognition

Example species details:
- Species name
- Category
- Habitat
- Conservation status
- Ecological description


### 🚨 Report Issue
Incident reporting system for sacred grove protection.

Users can report:
- Tree Cutting
- Waste Dumping
- Encroachment
- Illegal Hunting
- Path Obstruction
- Other violations

Features:
- Auto location fetch
- Google Maps location picker
- Camera capture
- Gallery upload
- Incident description
- Validation checks
- Submission confirmation notification


### 👤 User Authentication
Secure login system with:

- Admin login
- Create Account
- Google Sign-In
- Room Database user storage


### 🪪 Profile Management
Profile popup with logged-in user details:

- Name
- Email
- Access level
- Sign out functionality


## 🛠 Tech Stack
